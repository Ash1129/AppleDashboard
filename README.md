# Apple Bets on Augmented Reality Dashboard

A premium Apple-inspired single-page strategy dashboard for the Harvard Business School case study "Apple Bets on Augmented Reality."

## Contents

- Editorial single-page dashboard
- Responsive sections for market, ecosystem, competitors, facts, frameworks, timeline, and takeaways
- Floating case assistant with retrieval-based responses and inline mini charts for supported questions

## Files

- `index.html`: page shell
- `styles.css`: visual system, layout, responsive behavior, and chatbot styling
- `script.js`: component rendering, charts, navigation, and case assistant logic

## Preview

Open `index.html` directly in a browser or serve the folder locally, for example:

```bash
python3 -m http.server 4173
```
